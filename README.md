# Telegram Bot — Registration & Admin Panel

Telegram-бот на Go с полноценной системой регистрации и админкой для управления пользователями.

## Функционал

### Регистрация
- `/start` — регистрация нового пользователя
- Пользователь попадает в статус **pending** (ожидание)
- Админы получают уведомление с inline-кнопками для подтверждения/отклонения

### Админ-панель
- **📋 Профиль** — просмотр информации о текущем пользователе
- **👥 Управление** — список всех пользователей с кнопками управления
- **📥 Заявки** — список ожидающих подтверждения заявок

### Управление пользователями
| Действие | Описание |
|----------|----------|
| ✅ Одобрить | Подтвердить регистрацию пользователя |
| ❌ Отклонить | Отклонить заявку |
| 👑 Назначить админом | Повысить пользователя до админа |
| 👤 Снять админа | Вернуть роль обычного пользователя |
| 🚫 Заблокировать | Заблокировать аккаунт |
| 🔓 Разблокировать | Разблокировать аккаунт |
| 🗑 Удалить | Безвозвратно удалить аккаунт |

### Роли
- **superadmin** — создается автоматически из `.env`, полные права
- **admin** — управление пользователями
- **user** — обычный пользователь

### Статусы
- `pending` — ожидает подтверждения
- `approved` — активен
- `blocked` — заблокирован

## Технологии

- [Go](https://go.dev/)
- [telegram-bot-api](https://github.com/go-telegram-bot-api/telegram-bot-api) — Telegram Bot API
- [GORM](https://gorm.io/) + [SQLite](https://github.com/glebarez/sqlite) — ORM и база данных
- [godotenv](https://github.com/joho/godotenv) — загрузка `.env`

## Установка

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/YOUR_USERNAME/telegram-bot.git
cd telegram-bot
```

### 2. Установите зависимости

```bash
go mod tidy
```

### 3. Настройте конфигурацию

Создайте файл `.env` в корне проекта:

```env
TOKEN=YOUR_BOT_TOKEN_HERE
ADMIN_IDS=123456789,987654321
DB_PATH=bot.db
```

| Параметр | Описание |
|----------|----------|
| `TOKEN` | Токен бота от [@BotFather](https://t.me/BotFather) |
| `ADMIN_IDS` | Telegram ID суперадминов через запятую |
| `DB_PATH` | Путь к файлу базы данных SQLite |

> **Как узнать свой ID:** напишите боту [@userinfobot](https://t.me/userinfobot)

### 4. Запустите бота

```bash
go run main.go
```

Или соберите бинарник:

```bash
go build -o bot .
./bot
```

## Как это работает

1. Пользователь отправляет `/start` → создается запись в БД со статусом `pending`
2. Суперадмины получают уведомление с inline-кнопками
3. После подтверждения пользователь получает доступ к боту
4. Админы могут управлять пользователями через меню «Управление»
